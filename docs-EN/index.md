## Current Status

This repository is the initial scaffold for a `Unitree Go2 + FAST-LIO2 + Nav2` indoor museum guide project.

Already included:

- ROS 2 Humble workspace layout
- dependency manifest
- Go2 command bridge skeleton
- museum guide manager skeleton
- basic bringup config

Still pending:

- real Unitree SDK2 integration
- FAST-LIO2 hardware tuning
- Nav2 controller and costmap tuning
- full indoor localization validation
